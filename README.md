# NEXTINAOKA
TS boot campのアプリ  
- 期間：11/1 ~ 11/30  
- 概要：Webアプリを1から作成してデプロイまでやって、みんなで発表会する。  
- ルール：11/1~新しくコードを書き始める  
- フェーズ1：React+TS でWebアプリ作成
    - 開発期間：11/1 ~ 15
    - 発表会：15日
- フェーズ2：React+TSでLIFFアプリ作成
    - 開発期間：11/16 ~ 30
    - 発表会：30日

# 使い方
クローンからローカルサーバー起動まで
```
git clone 
yarn install
touch next-env.d.ts
yarn dev
```
eslintとprettier
```
yarn lint
yarn format
```
